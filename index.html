Strong

(voltar)

Usar strong entre <> serve para deixar uma palavra em negrito, conforme o exemplo: negrito.

A

(voltar)

Usar "a" com um < serve para criar um link que direciona para outra página. No exemplo, o link a seguir vai direcionar para o buscador Google. Usando "title" pode nomear o link, e usando "target=blank" o link abre em uma nova página.

Mark

(voltar)

Mark serve para marcar determinado texto em amarelo, conform a palavra a seguir: texto marcado.

Small

(voltar)

Small serve para deixar um texto com letra menor, conforme a seguir: voltar.

Blockquote

(voltar)

Blockquote serve para deixar o texto um pouco afastado da margem, como se fosse uma citação:

"Penso, logo existo"

Ol

(voltar)

OL significa "lista ordenada", e cria uma lista que numera os itens:

  1. item a
  2. item b
  3. item c

Ul

(voltar)

UL significa "lista sem ordem", e cria uma lista que não numera os itens, apenas os lista com pontos:

  • item a
  • item b
  • item c

U

(voltar)

U entre <> serve para usar o underline, ou a linha abaixo de uma palavra ou frase, conforme a seguir: frase sublinhada.

I

(voltar)

I entre <> serve para deixar uma palavra ou frase em itálico, ou seja, levemente inclinada: frase em itálico.
